# d3Mol - React D3 molecule
## steps to run
1. download caseManager.zip from https://github.com/tetsuyac/d3Mol.git
1. unzip d3Mol.zip
1. cd d3Mol
1. npm install
1. npm start
1. visit http://localhost:3010 with local browser
1. have a fun!
